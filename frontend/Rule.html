Patient Management System

Application Usage Rules & Policy

1. User Authorization

  • Only registered users can access the system.
  • Users must log in using valid credentials.
  • Login details must not be shared.
  • Access is granted based on user roles (Admin, Doctor, Staff).

2. Patient Data Handling

  • Create a new patient enter data.
  • All patient data must be accurate and verified.
  • Duplicate patient records are not allowed.
  • Unauthorized modification of records is prohibited.
  • All changes must be properly logged.

3. Data Privacy & Security

  • Patient information is strictly confidential.
  • Data must not be shared without proper authorization.
  • Users must log out after completing their work.
  • System activity may be monitored for security purposes.

4. System Restrictions

  • The system must be used only for medical purposes.
  • Storing personal or unrelated data is prohibited.
  • Unauthorized access attempts will lead to account suspension.

5. Backup & Compliance

  • Regular data backups must be maintained.
  • Violation of rules may result in termination of access.
